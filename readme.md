# 《生-成》
持存、镶嵌与流变：基于区分的属性逻辑过程

## 关于本书
《生-成》是一套以「可区分性」为元公理构建的原创生成论哲学体系。它跳出了西方传统存在论的框架，从最底层的“区分”出发，重新定义了世界的生成方式，为理解自然、意识与人类文明提供了一套全新的底层语法。

## 核心观点
- 区分是生成的基础
- 持存是生成结果的筛选
- 世界是属性的
- 识与结构合一

## 三种存在方式
1.  **持存**：生成结构的稳定延续
2.  **镶嵌**：生成结构之间的相互关联
3.  **流变**：生成结构的持续更新
