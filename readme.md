# 《生-成》
持存、镶嵌与流变：基于区分的属性逻辑过程

## 关于本书:
《生-成》是一套以「区分事件」为基础构建的原创生成论哲学体系。它跳出了西方传统存在论的框架，从最底层的“区分”出发，重新定义了世界的生成方式，为理解自然、意识与人类文明提供了一套全新的底层语法。

## 核心观点:
- 区分是生成的基础
- 持存是生成结果的筛选
- 世界是属性的
- 识与结构合一

## 三种存在方式:
1.  **持存**：生成结构的稳定延续
2.  **镶嵌**：生成结构之间的相互关联
3.  **流变**：生成结构的持续更新

## 摘要:
	本书无意凿刻某个不容置疑的 “真理” 丰碑，而是试图铺展一幅基于 “空” 的广袤认知图景，从最原初的区分事件开始，一步步推演出属性如何耦合、逻辑如何展开、过程如何运转，最终凝练出一套“从无限不确定背景中生成有限确定性”的生成方法：它如同暗夜灯塔，指引我们在不确定的迷雾中捕捉更多的可能，更好地把握 “万有” 之确定性……为我们维持系统的持存与演化提供了统一的解释框架与实践路径。
